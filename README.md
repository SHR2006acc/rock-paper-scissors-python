# Rock Paper Scissors – Python

Simple command-line **Rock Paper Scissors game** written in Python.

## Features

* User vs AI
* Random AI moves
* Clear terminal interface
* Replay option

## How it works

The user selects:
1 → Rock
2 → Paper
3 → Scissors

The AI randomly selects a move and the program determines the winner.

## How to Run

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/rock-paper-scissors-python.git
```

Run the program:

```
python rock_paper_scissors.py
```

## Example

```
Welcome to Rock-Paper-Scissors

1 - Rock

3 - Scissors

User : rock
```
